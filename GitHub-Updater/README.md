# 🚀 GitHub Auto Updater - מעדכן אוטומטי לגיטהאב

מערכת שמעדכנת אוטומטית את **כל ה-repository** לגיטהאב עם כל שינוי שאתה עושה!

## ✨ **תכונות חדשות**

- 🔍 **אוטומציה למציאת פורט** - המערכת מוצאת פורט פנוי אוטומטית
- 🚀 **הפעלה מהירה** - קובץ הפעלה אחד עושה הכל
- 📊 **לוגים מפורטים** - רואה בדיוק מה קורה
- 🔄 **עדכון אוטומטי** - מזהה שינויים ועדכן אוטומטית

## 🎯 **איך להפעיל**

### **אפשרות 1: הפעלה חכמה (מומלץ)**
```bash
הפעל-שרת-חכם.bat
```

### **אפשרות 2: הפעלה מהירה**
```bash
הפעל-מהיר.bat
```

### **אפשרות 3: ידני**
```bash
npm install
node github-updater-server.js
```

## 📋 **מה המערכת עושה**

1. **🔍 מוצאת פורט פנוי** - בודקת פורטים 3001-3005, 8080-8082, 5000-5001
2. **🚀 מפעילה שרת** - שרת Node.js עם Express
3. **📊 בודקת מערכת** - Git, Repository, הרשאות גיטהאב
4. **🔄 מזהה שינויים** - בודקת כל 30 שניות
5. **📤 מעדכנת גיטהאב** - מעלה שינויים אוטומטית

## 🌐 **גישה לאפליקציה**

לאחר הפעלת השרת, פתח דפדפן בכתובת:
```
http://localhost:XXXX/github-auto-updater.html
```
(XXXX הוא הפורט שנמצא אוטומטית)
